Son herramientas que se utilizan para la integracion continua dentro del desarollo de software
automatizan tareas como compilacion, pruebas y despliegue 

la integracion contnua (CI)
la integracion continua es la automatizacion del proceso de prueba,en paso el codigo o cambios tengan error retornaran y
para que sean solicionados y nuevamente se suabn los cambios para hacer el proceso de lanzamientos.
En resumen 
los cambios en el codigo se van realizando de autmatica, En el (CI) se realiza toda la parte de pruebas antes de compilar 

Entrega continua (CD)
Este es proceso despues del (CI) en el que el codigo ya a tenido que psar las pruebas para poder ser lanzado a produccion como una nueva mejora 
en este proceso se realiza la compilacion y ejecucion del codigo.
